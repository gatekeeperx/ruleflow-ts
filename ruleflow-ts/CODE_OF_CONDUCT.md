# Code of Conduct

We are committed to fostering a welcoming and inclusive community.

- Be respectful and considerate.
- No harassment, discrimination, or hateful conduct.
- Use inclusive and professional language.
- Accept constructive feedback graciously.

If you witness or experience unacceptable behavior, please report it via GitHub Issues or by contacting the maintainers.

Consequences for unacceptable behavior may include temporary or permanent bans from community spaces.

This Code of Conduct is adapted from the Contributor Covenant.
